# Camera Mount

Mount the camera so it has a clear view of the projected surface. The camera should see the same area the projector covers (or a known subset) for calibration to work.

- **stl/** — Printable STL files for common mounts.
- **cad/** — Source CAD for modifications.

Align the camera to avoid obstructions and minimize glare from the projector. Height and angle depend on your setup (table vs wall projection).
