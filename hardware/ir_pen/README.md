# IR Pen

Build or source an IR pen whose tip is visible to an IR-sensitive camera. The LED should be at the tip and powered (battery or USB).

- **pen_bom.csv** — Parts list for a DIY pen.
- **wiring_diagram.svg** — Wiring reference (template only; replace or augment with your schematic for the LED, switch, and power per pen_bom.csv).

For visible bright-point mode you can use a laser pointer or bright LED instead; no IR hardware needed.
